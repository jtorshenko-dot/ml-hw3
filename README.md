# ML Homework 3

Проект, посвящённый базовой генерации признаков для табличного машинного обучения. В работе рассматриваются методы предобработки данных, feature engineering, оценка качества классификации и построение ML-пайплайна на данных матчей Dota 2.

Ноутбук включает работу с табличными данными, кодирование признаков, временные разбиения выборки, валидацию моделей и оценку качества с помощью ROC-AUC и коэффициента Джини. Для анализа используются инструменты pandas, numpy и scikit-learn, а сама работа ориентирована на практику построения устойчивых признаков для задачи бинарной классификации.

Проект показывает базовые навыки feature engineering, model validation и подготовки данных для прикладных ML-задач.
